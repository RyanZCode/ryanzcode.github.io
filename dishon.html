Dishon Data Systems

Use the buttons at the top of the page to navigate this web app.

The Machine Monitoring System uses real-time data from PowerRadar™ combined with historical data trends to determine the status of each machine and provide an real-time overview of machine uptime. This system displays the current status of each machine (in cycle, idle, or off) and its percentage uptime for the past 24 hours.
Lathes/Millturn Monitoring Mill4ax/Mill5ax Monitoring Grinding Monitoring All Machines Monitoring

The Quality Queue System uses data from Abas ERP along with a data algorithm to determine if a work order is in quality. The system is a real-time display of the current quality status of each machine. This system shows all the current open work orders that are in quality.
The MRB Tracker System uses data from Abas ERP to display all the current open MRB listings.
Quality Queue MRB Tracker

The WIP Report System displays data from Abas ERP to show current work orders in progress.
WIP Report

The Yield Notification System delivers daily yield notifications containing data from Abas ERP.
The Processing Notification System delivers daily processing notifications containing data from Abas ERP.
The Materials Notification System delivers daily materials notifications containing data from Abas ERP.

More data systems may be added in the future. To request a new data system to be developed, or provide feedback on existing systems, please use the feedback form below.
